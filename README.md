# Abuzar Gifary - Personal Portfolio

A clean, responsive, and modern personal portfolio website built using HTML, Tailwind CSS, and JavaScript.

## Features
- **Responsive Design:** Mobile-first design optimized for all screen sizes.
- **Dark Mode:** Easily toggle between light and dark visual themes.
- **Featured Projects:** Showcases web and mobile apps, including projects from **ByteGif**.
- **Contact Form:** Integrated contact form utilizing Web3Forms for direct messages.

## Tech Stack
- HTML5
- Tailwind CSS (via Play CDN)
- JavaScript
